# Kaodim Automated Test Scripts

[![CircleCI](https://circleci.com/gh/Kritig02/kaodim_automated_scripts.svg?style=svg)](https://circleci.com/gh/Kritig02/kaodim_automated_scripts)

Supported Browsers : 
1. Chrome (Headless)
2. Firefox (Gecko)

## Installation : 

```
    $ npm install
```

## Running Tests : 

```
    $ npm test
```

### Testing on Firefox : 

```
    $ npm run firefox
```
